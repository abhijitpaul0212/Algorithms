### 1-0-4) Reverse String

```java
private String reverseString(String input){

    if (input.equals(null) || input.length() == 0){
        return null;
    }

    StringBuilder builder = new StringBuilder(input).reverse();
    return builder.toString();
} 
```

```java
        chars[y] = temp;
    }
```   
